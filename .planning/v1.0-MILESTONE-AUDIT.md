## Defect Fixed During Audit

**`curl` absent from `python:3.11-slim` healthcheck** (Critical → FIXED)

`docker-compose.yml` backend healthcheck used `["CMD", "curl", ...]` but `python:3.11-slim` doesn't ship `curl`. Container would have been marked `unhealthy` after 5 retries.

**Fix applied:** Replaced with Python urllib probe (no extra dependencies):
```yaml
test: ["CMD-SHELL", "python -c \"import urllib.request; urllib.request.urlopen('http://localhost:8000/health')\""]
```
