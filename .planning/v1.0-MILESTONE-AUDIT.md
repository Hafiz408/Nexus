### Critical Integration Defect ✗

**`curl` absent from python:3.11-slim (INFRA-01/INFRA-02)**

The backend healthcheck in `docker-compose.yml`:
```yaml
test: ["CMD", "curl", "-f", "http://localhost:8000/health"]
```

`python:3.11-slim` does not include `curl`. The Dockerfile has no `apt-get install curl`. Every healthcheck probe will fail with `curl: not found` → container marked `unhealthy` after 5 retries.

**Fix option A** (add curl to image):
```dockerfile
RUN apt-get update && apt-get install -y --no-install-recommends curl && rm -rf /var/lib/apt/lists/*
```

**Fix option B** (Python probe, no extra dependency):
```yaml
test: ["CMD", "python", "-c", "import urllib.request; urllib.request.urlopen('http://localhost:8000/health')"]
```
