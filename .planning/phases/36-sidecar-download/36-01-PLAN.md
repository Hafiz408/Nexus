**Method 1: `_fetchChecksum(baseUrl: string, archiveName: string): Promise<string>`**

- Fetches `${baseUrl}/checksums.sha256` using global `fetch`
- If `!res.ok`, throws `new Error(\`Failed to fetch checksums: HTTP ${res.status}\`)`
- Parses the response text line by line: split on `'\n'`, trim each line, skip empty lines
- For each line, find first space with `trimmed.indexOf(' ')`, extract hash as `trimmed.slice(0, spaceIdx)`, extract filename as `trimmed.slice(spaceIdx).trim()`
- If `filename === archiveName`, return the hash string
- If no match found after all lines, throw `new Error(\`No checksum found for ${archiveName} in checksums.sha256\`)`
- Log to `this._channel.appendLine(\`[SidecarManager] Fetching checksum for ${archiveName}...\`)` at the start

**Method 2: `_downloadAndVerify(url: string, destPath: string, expectedHash: string, progress: vscode.Progress<{ message?: string; increment?: number }>): Promise<void>`**

- Fetches the archive from `url` using global `fetch`
- If `!res.ok`, throws `new Error(\`Download failed: HTTP ${res.status}\`)`
- If `!res.body`, throws `new Error('Response body is null')`
- Reads `Content-Length` header: `const contentLength = parseInt(res.headers.get('content-length') ?? '0', 10);`
- Creates incremental hash: `const hash = crypto.createHash('sha256');`
- Declares `const chunks: Uint8Array[] = []; let received = 0; let lastPct = 0;`
- Iterates body stream: `for await (const chunk of res.body as AsyncIterable<Uint8Array>)`
  - Calls `hash.update(chunk)` and `chunks.push(chunk)`
  - Increments `received += chunk.length`
  - If `contentLength > 0`: calculates `const pct = Math.floor((received / contentLength) * 100)`, and if `pct > lastPct`, calls `progress.report({ increment: pct - lastPct })` then sets `lastPct = pct`
- After loop: `const digest = hash.digest('hex');`
- If `digest !== expectedHash`:
  - Calls `fs.rmSync(destPath, { force: true })` (delete corrupt file per CONTEXT.md decision)
  - Throws `new Error(\`SHA256 mismatch: expected ${expectedHash}, got ${digest}\`)`
- On success: `fs.writeFileSync(destPath, Buffer.concat(chunks));`
- Logs `this._channel.appendLine(\`[SidecarManager] Download complete: ${received} bytes, SHA256 verified.\`)` at the end

**Method 3: `_showDownloadError(errMsg: string): Promise<void>`**

- Calls `const action = await vscode.window.showErrorMessage(\`Nexus: Failed to download backend — ${errMsg}. Download manually or check your network.\`, 'Open GitHub Releases');`
- If `action === 'Open GitHub Releases'`, calls `await vscode.env.openExternal(vscode.Uri.parse('https://github.com/Hafiz408/Nexus/releases'));`
