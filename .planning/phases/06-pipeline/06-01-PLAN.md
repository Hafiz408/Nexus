  <action>
    Create backend/app/ingestion/pipeline.py. This is a new file. Implement exactly as follows:

    IMPORTS (in this order):
    ```python
    import asyncio
    import logging
    from asyncio import Semaphore

    from app.ingestion.walker import walk_repo, EXTENSION_TO_LANGUAGE
    from app.ingestion.ast_parser import parse_file
    from app.ingestion.graph_builder import build_graph
    from app.ingestion.embedder import embed_and_store
    from app.ingestion.graph_store import save_graph, delete_nodes_for_files
    from app.models.schemas import IndexStatus
    ```

    MODULE CONSTANTS:
    ```python
    logger = logging.getLogger(__name__)
    PARSE_CONCURRENCY = 10
    _status: dict[str, IndexStatus] = {}
    ```

    FUNCTION: get_status(repo_path: str) -> IndexStatus | None
    - Returns _status.get(repo_path)

    FUNCTION: async _parse_concurrent(files: list[dict], repo_path: str) -> tuple[list, list]
    - Creates Semaphore(PARSE_CONCURRENCY)
    - Inner async def _one(entry: dict): uses "async with sem:" then "await asyncio.to_thread(parse_file, entry["path"], repo_path, entry["language"])"
    - Runs: results = await asyncio.gather(*[_one(e) for e in files], return_exceptions=True)
    - Filters results: if isinstance(r, Exception): logger.warning("parse_file failed: %s", r); continue; else nodes, edges = r; all_nodes.extend(nodes); all_edges.extend(edges)
    - Returns (all_nodes, all_edges)

    FUNCTION: async run_ingestion(repo_path: str, languages: list[str], changed_files: list[str] | None = None) -> IndexStatus
    - Sets _status[repo_path] = IndexStatus(status="running")
    - Wraps entire body in try/except Exception as exc
    - INCREMENTAL PATH (when changed_files is not None):
      - Calls delete_nodes_for_files(changed_files, repo_path)
      - Builds ext_map: {ext.lstrip("."): lang for ext, lang in EXTENSION_TO_LANGUAGE.items()}
      - Builds files_to_parse from changed_files: include entry only if "." in f AND suffix (f.rsplit(".", 1)[-1].lower()) is in ext_map AND ext_map[suffix] is in languages
      - Entry format: {"path": f, "language": ext_map[suffix], "size_kb": 0}
    - FULL PATH (when changed_files is None):
      - files_to_parse = walk_repo(repo_path, languages)
    - Updates _status[repo_path] = IndexStatus(status="running", files_processed=len(files_to_parse))
    - Calls all_nodes, all_edges = await _parse_concurrent(files_to_parse, repo_path)
    - Calls G = build_graph(all_nodes, all_edges)
    - Calls await asyncio.to_thread(save_graph, G, repo_path)
    - Calls nodes_stored = await asyncio.to_thread(embed_and_store, all_nodes, repo_path)
    - Builds result = IndexStatus(status="complete", nodes_indexed=nodes_stored, edges_indexed=G.number_of_edges(), files_processed=len(files_to_parse))
    - except block: logger.exception("run_ingestion failed for %s", repo_path); result = IndexStatus(status="failed", error=str(exc))
    - Sets _status[repo_path] = result
    - Returns result

    CRITICAL: embed_and_store and save_graph MUST be wrapped in asyncio.to_thread (they are blocking I/O). Do NOT call them directly in the async function body.
    CRITICAL: asyncio.gather MUST use return_exceptions=True so a single failed parse does not cancel all other parses.
    CRITICAL: Do NOT call asyncio.run() anywhere in this file — run_ingestion is itself async and will be awaited by callers.
  </action>
