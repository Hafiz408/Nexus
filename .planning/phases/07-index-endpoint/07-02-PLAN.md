    Replace the entire contents of `backend/app/main.py` with:

    ```python
    from contextlib import asynccontextmanager

    from fastapi import FastAPI
    from fastapi.middleware.cors import CORSMiddleware

    from app.api.index_router import router as index_router
    from app.db.database import init_db
    from app.ingestion.embedder import init_pgvector_table


    @asynccontextmanager
    async def lifespan(app: FastAPI):
        init_db()
        init_pgvector_table()
        yield


    app = FastAPI(title="Nexus API", version="1.0.0", lifespan=lifespan)

    # CORS must be registered before include_router — middleware wraps the full app stack
    app.add_middleware(
        CORSMiddleware,
        allow_origins=["http://localhost:3000"],
        allow_origin_regex=r"vscode-webview://.*",
        allow_methods=["GET", "POST", "DELETE"],
        allow_headers=["*"],
    )

    app.include_router(index_router)


    @app.get("/health")
    def health():
        return {"status": "ok", "version": "1.0.0"}
    ```

    Critical notes:
    - CORSMiddleware is added BEFORE include_router. In Starlette, middleware wraps the entire application stack — registration order matters. Adding it after routers may cause CORS headers to be missing on some preflight responses.
    - Do NOT set `allow_credentials=True`. Combining wildcard regex (`allow_origin_regex`) with `allow_credentials=True` causes browsers to reject the CORS response.
    - `allow_origin_regex=r"vscode-webview://.*"` — the `r""` raw string is required so the backslash in `.*` is not consumed by Python string processing.
    - `allow_origins=["http://localhost:3000"]` — plain string matching for the dev server origin.
    - The existing lifespan, init_db, and init_pgvector_table lines are preserved unchanged.
