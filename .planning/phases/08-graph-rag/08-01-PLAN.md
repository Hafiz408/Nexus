    **Function 1 — semantic_search (RAG-01):**
    Signature: `def semantic_search(query: str, repo_path: str, top_k: int) -> list[tuple[str, float]]`
    - Instantiate OpenAI client lazily inside the function body: `client = OpenAI(api_key=get_settings().openai_api_key)` (same lazy pattern as embedder.py — prevents ValidationError on import)
    - Call `client.embeddings.create(model="text-embedding-3-small", input=[query])`
    - Get `query_vec = response.data[0].embedding`
    - Open connection: `conn = get_db_connection()`, call `register_vector(conn)` per-connection (Phase 05 decision)
    - Query: `SELECT id, 1 - (embedding <=> %s::vector) AS score FROM code_embeddings WHERE repo_path = %s ORDER BY embedding <=> %s::vector LIMIT %s` with params `(query_vec, repo_path, query_vec, top_k)`
    - Return `list[tuple[str, float]]` — node_id + score pairs only (NOT CodeNode objects — code_embeddings table lacks signature/docstring/body_preview; Phase 8 research pitfall 1)
    - Always call `conn.close()` in a finally block

    **Function 2 — expand_via_graph (RAG-02):**
    Signature: `def expand_via_graph(seed_node_ids: list[str], G: nx.DiGraph, hop_depth: int, edge_types: list[str] | None = None) -> set[str]`
    - If edge_types is not None, create a filtered subgraph view FIRST: `G_work = nx.subgraph_view(G, filter_edge=lambda u, v: G[u][v].get("type") in edge_types)` — zero-copy view (Phase 8 research pitfall 3)
    - Otherwise `G_work = G`
    - For each seed_node_id: guard with `if node_id not in G_work: logger.warning("seed node %s not in graph, skipping", node_id); continue`
    - Call `subgraph = nx.ego_graph(G_work, node_id, radius=hop_depth, undirected=True)` — covers BOTH predecessors and successors; DO NOT use nx.bfs_tree (only follows outgoing edges)
    - Collect `expanded.update(subgraph.nodes())`
    - Return deduplicated `set[str]`

    **Function 3 — rerank_and_assemble (RAG-03):**
    Signature: `def rerank_and_assemble(expanded_node_ids: set[str], seed_scores: dict[str, float], G: nx.DiGraph, max_nodes: int) -> list[CodeNode]`
    - Compute max in_degree for normalisation: `in_degrees = [G.nodes[n].get("in_degree", 0) for n in expanded_node_ids if n in G]`; `max_in_degree = max(in_degrees) if in_degrees else 1` — guard against zero-division
    - For each node_id in expanded_node_ids: skip if not in G
    - Read attrs: `attrs = G.nodes[node_id]`
    - Apply EXACT formula from RAG-03 verbatim: `semantic = seed_scores.get(node_id, 0.3)` (0.3 fallback for non-seed nodes); `pagerank = attrs.get("pagerank", 0.0)`; `in_degree_norm = attrs.get("in_degree", 0) / max_in_degree`; `score = semantic + (0.2 * pagerank) + (0.1 * in_degree_norm)`
    - Reconstruct CodeNode from graph attributes: `CodeNode(**{k: v for k, v in attrs.items() if k in CodeNode.model_fields})` — graph stores full model_dump from pipeline
    - Sort by score descending, return top `max_nodes` CodeNode objects

    **Function 4 — graph_rag_retrieve (RAG-04):**
    Signature: `def graph_rag_retrieve(query: str, repo_path: str, G: nx.DiGraph, max_nodes: int = 10, hop_depth: int = 1) -> tuple[list[CodeNode], dict]`
    - Step 1: `seed_results = semantic_search(query, repo_path, top_k=max_nodes)`; `seed_scores = {node_id: score for node_id, score in seed_results}`
    - Step 2: `expanded = expand_via_graph(list(seed_scores.keys()), G, hop_depth)`
    - Step 3: `nodes = rerank_and_assemble(expanded, seed_scores, G, max_nodes)`
    - Stats dict: `{"seed_count": len(seed_scores), "expanded_count": len(expanded), "returned_count": len(nodes), "hop_depth": hop_depth}`
    - Return `(nodes, stats)`
