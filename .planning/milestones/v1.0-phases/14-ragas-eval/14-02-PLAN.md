HELPERS:

async def get_answer(nodes: list[CodeNode], question: str) -> str:
  Collects all tokens from explore_stream() into a single string and returns it.

def naive_retrieve(question: str, repo_path: str, G: nx.DiGraph, max_nodes: int = 10) -> list[CodeNode]:
  Calls semantic_search(question, repo_path, top_k=max_nodes), then hydrates CodeNode objects from G.nodes[node_id] for each (node_id, score) pair where node_id in G. Wraps hydration in try/except to skip malformed nodes. Returns list[CodeNode].

def build_contexts(nodes: list[CodeNode]) -> list[str]:
  Returns list of formatted context strings per node: "{n.file_path}:{n.line_start}-{n.line_end}\n{n.signature}\n{n.docstring or ''}\n{n.body_preview}".

MAIN ASYNC FUNCTION: async def main(repo_path: str) -> None

Step 1 — Load graph and golden dataset:
  - Call get_settings() to load config
  - Call load_graph(repo_path) → G; print node/edge count
  - Load eval/golden_qa.json using os.path.abspath(__file__) to find it regardless of cwd

Step 2 — Build sample lists:
  - Loop over golden_qa entries, print progress "[i/30] Q0N: first 60 chars..."
  - For each entry: run graph_rag_retrieve(question, repo_path, G, max_nodes=10, hop_depth=1) → graph_nodes
  - Call await get_answer(graph_nodes, question) → graph_response
  - Build graph SingleTurnSample(user_input=question, retrieved_contexts=build_contexts(graph_nodes), response=graph_response, reference=reference)
  - For naive: call naive_retrieve(question, repo_path, G, max_nodes=10) → naive_nodes
  - Call await get_answer(naive_nodes, question) → naive_response
  - Build naive SingleTurnSample with same fields

Step 3 — Set up RAGAS evaluator:
  - evaluator_llm = LangchainLLMWrapper(ChatOpenAI(model="gpt-4o-mini", temperature=0, api_key=settings.openai_api_key))
  - evaluator_embeddings = LangchainEmbeddingsWrapper(OpenAIEmbeddings(model="text-embedding-3-small", api_key=settings.openai_api_key))
  - metrics = [Faithfulness(llm=evaluator_llm), ResponseRelevancy(llm=evaluator_llm, embeddings=evaluator_embeddings), ContextPrecision(llm=evaluator_llm)]
  - run_config = RunConfig(timeout=120, max_retries=3)
  - NOTE: Always pass llm= explicitly. Never use metric singletons (old API).

Step 4 — Evaluate both modes:
  - Create timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
  - Create eval/results/ via os.makedirs(..., exist_ok=True)
  - For mode, samples in [("graph_rag", graph_samples), ("naive_vector", naive_samples)]:
    - Build EvaluationDataset(samples=samples)
    - Call evaluate(dataset=dataset, metrics=metrics, run_config=run_config, show_progress=True, raise_exceptions=False)
    - Call results.to_pandas() → df
    - Use a col_map dict to find actual column names: iterate df.columns, match against ("faithfulness", "answer_relevancy", "response_relevancy", "context_precision") using `key in col.lower()`. This handles column name variation between ragas 0.4.x minor versions.
    - Define safe_mean(col_key) that returns float(df[col].mean()) or None if col missing/all-NaN
    - Build aggregate = {"faithfulness": ..., "answer_relevancy": safe_mean("answer_relevancy") or safe_mean("response_relevancy"), "context_precision": ...}
    - Build output dict: {"timestamp", "mode", "repo_path", "aggregate", "per_question": df.to_dict(orient="records")}
    - Write to eval/results/ragas_results_{mode}_{timestamp}.json using json.dump(..., default=str)
    - Track all_aggregates[mode] = aggregate

Step 5 — Write comparison file:
  - Build comparison dict: {"timestamp", "repo_path", "graph_rag": all_aggregates["graph_rag"], "naive_vector": all_aggregates["naive_vector"], "winner": {metric: "graph_rag" or "naive_vector" for each metric where both values non-None}}
  - Write to eval/results/ragas_comparison_{timestamp}.json
  - Print comparison summary table to stdout showing metric | graph_rag | naive_vector | winner

PITFALLS TO AVOID (from RESEARCH.md):
- Do NOT use `from ragas.metrics import faithfulness` (lowercase singleton) — raises ImportError in 0.4.x
- Do NOT call asyncio.run() inside the evaluation loop — wrap entire loop in async def main()
- Do NOT forget raise_exceptions=False — otherwise a single bad API response aborts all 30 questions
- Do NOT use json.dump without default=str — pandas floats may be numpy.float64 which is not JSON-serializable
- DO use RunConfig(timeout=120, max_retries=3) to survive rate limits
