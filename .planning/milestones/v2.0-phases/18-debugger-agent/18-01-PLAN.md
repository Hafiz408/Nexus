**Constants (module-level):**

```python
ERROR_KEYWORDS = frozenset({"try", "except", "raise", "catch", "throw", "error", "exception"})

DEBUGGER_SYSTEM = """You are a code debugging assistant. Given a bug description and a list
of suspect functions with anomaly scores, generate a concise diagnosis narrative (2-4 sentences).

CRITICAL: Only mention function names from this list: {traversal_names}
Do NOT hallucinate function names not in this list."""

DEBUGGER_PROMPT = ChatPromptTemplate.from_messages([
    ("system", DEBUGGER_SYSTEM),
    ("human", "Bug: {question}\n\nSuspects (ranked by anomaly score):\n{suspects_text}"),
])
```

**Private helpers:**

`_find_entry_nodes(question: str, G: nx.DiGraph) -> list[str]`:
- Lowercase the question
- For each node_id in G.nodes(): get `name` attr, check if name is non-empty and `name.lower() in question_lower`
- Return list of matching node_ids
- Fallback handled in debug() — this function returns empty list on no match (never raises)

`_forward_bfs(G: nx.DiGraph, entry_id: str, max_hops: int) -> list[str]`:
- Use `deque([(entry_id, 0)])`; `seen = {entry_id}`; `visited = []`
- At each step: if depth > 0, append node_id to visited (entry itself excluded from result per open question resolution — include it at position 0 so isolated entry nodes still appear as suspects; actually per research recommendation: include entry in traversal_path)
- CORRECTION: Include entry node in visited only at depth 0 as the starting point — push it to the traversal list so isolated entry nodes get scored. Specifically: append node to visited for ALL depths (including 0). This ensures that if `entry -> nothing` the entry still appears.
- Filter edges: `G.edges[u, v].get("type") == "CALLS"` (use `.get()` not `[]` to avoid KeyError)
- Stop expanding at `depth >= max_hops`

`_score_node(attrs: dict, question_tokens: set[str]) -> float`:
- Factor 1 (complexity, weight 0.30): `min(attrs.get("complexity", 1) / 10.0, 1.0)`
- Factor 2 (error handling absence, weight 0.25): Build body string from `body_preview` + `docstring`. `f_error = 0.0 if any(kw in body_lower for kw in ERROR_KEYWORDS) else 1.0`
- Factor 3 (keyword match, weight 0.20): `body_tokens = set(re.findall(r"\w+", body_lower))`; `overlap = len(question_tokens & body_tokens)`; `f_keyword = min(overlap / max(len(question_tokens), 1), 1.0)`
- Factor 4 (out-degree coupling, weight 0.15): `min(attrs.get("out_degree", 0) / 10.0, 1.0)`
- Factor 5 (inverted PageRank, weight 0.10): `pagerank = attrs.get("pagerank", 0.0)`; `f_pagerank = 1.0 - min(pagerank * 5.0, 1.0)`
- Final: `score = 0.30*f1 + 0.25*f2 + 0.20*f3 + 0.15*f4 + 0.10*f5`; `return min(max(score, 0.0), 1.0)`
- Weights sum to 1.0; each individual factor is already clamped to [0.0, 1.0] before weighting

`_build_reasoning(attrs: dict, score: float, question_tokens: set[str]) -> str`:
- Return a human-readable one-sentence reasoning string
- Example: `f"complexity={attrs.get('complexity',1)}, {'no error handling' if score > 0.5 else 'has error handling'}, anomaly_score={score:.2f}"`

`_impact_radius(G: nx.DiGraph, top_suspect_id: str) -> list[str]`:
- Return `[pred for pred in G.predecessors(top_suspect_id) if G.edges[pred, top_suspect_id].get("type") == "CALLS"]`

**Signature:** `def debug(question: str, G: nx.DiGraph, settings=None) -> DebugResult:`

**Step-by-step implementation:**

1. **Settings (lazy import):**
   ```python
   if settings is None:
       from app.config import get_settings  # noqa: PLC0415
       settings = get_settings()
   max_hops = settings.debugger_max_hops  # int, default 4 from config.py
   ```

2. **Entry node discovery:**
   ```python
   entry_nodes = _find_entry_nodes(question, G)
   if not entry_nodes and G.number_of_nodes() > 0:
       # Fallback: node with highest in_degree (most-called); ties broken by node_id sort
       entry_nodes = [max(G.nodes(), key=lambda n: G.nodes[n].get("in_degree", 0))]
   ```

3. **Forward BFS traversal:**
   ```python
   traversal: list[str] = []
   for entry in entry_nodes:
       traversal.extend(_forward_bfs(G, entry, max_hops))
   traversal = list(dict.fromkeys(traversal))  # deduplicate, preserve BFS order
   ```

4. **Score each node in traversal:**
   ```python
   question_tokens = set(re.findall(r"\w+", question.lower()))
   scored: list[tuple[str, float, dict]] = []
   for node_id in traversal:
       if node_id not in G:
           continue
       attrs = dict(G.nodes[node_id])
       score = _score_node(attrs, question_tokens)
       scored.append((node_id, score, attrs))
   ```

5. **Rank top 5:**
   ```python
   scored.sort(key=lambda x: x[1], reverse=True)
   top5 = scored[:5]
   ```

6. **Impact radius from top suspect:**
   ```python
   impact: list[str] = []
   if top5:
       impact = _impact_radius(G, top5[0][0])
   ```

7. **Build SuspectNode list:**
   ```python
   suspects = [
       SuspectNode(
           node_id=nid,
           file_path=attrs.get("file_path", ""),
           line_start=int(attrs.get("line_start", 0)),
           anomaly_score=score,
           reasoning=_build_reasoning(attrs, score, question_tokens),
       )
       for nid, score, attrs in top5
   ]
   ```

8. **LLM diagnosis narrative (lazy import — CRITICAL: must be inside function body):**
   ```python
   from app.core.model_factory import get_llm  # noqa: PLC0415
   llm = get_llm()
   traversal_names = [G.nodes[n].get("name", n) for n in traversal if n in G]
   suspects_text = "\n".join(
       f"{i+1}. {s.node_id} (score={s.anomaly_score:.2f}): {s.reasoning}"
       for i, s in enumerate(suspects)
   ) or "No suspects identified."
   prompt = DEBUGGER_PROMPT.partial(traversal_names=", ".join(traversal_names) or "none")
   chain = prompt | llm
   response = chain.invoke({"question": question, "suspects_text": suspects_text})
   diagnosis = response.content if hasattr(response, "content") else str(response)
   ```

9. **Return:**
   ```python
   return DebugResult(
       suspects=suspects,
       traversal_path=traversal,
       impact_radius=impact,
       diagnosis=diagnosis,
   )
   ```

**Key invariants to maintain:**
- `get_llm()` import MUST be inside debug() body — never at module level
- `get_settings()` import MUST be inside debug() body — never at module level

- Final score MUST be clamped: `min(max(score, 0.0), 1.0)`
