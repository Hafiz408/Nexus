Helper function:
```python
def _make_mock_llm(intent: str, confidence: float) -> MagicMock:
    mock_structured = MagicMock()
    mock_structured.invoke.return_value = IntentResult(
        intent=intent, confidence=confidence, reasoning="mock reasoning"
    )
    mock_llm = MagicMock()
    mock_llm.with_structured_output.return_value = mock_structured
    return mock_llm
```

Fixture:
```python
@pytest.fixture
def mock_llm_factory():
    with patch("app.agent.router.get_llm") as mock_factory:
        yield mock_factory
```
