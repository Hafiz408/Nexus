# v4.1 AV-Safe Binary Distribution — Archive (Shipped: 2026-04-02)

## Milestone Goal
Move the PyInstaller backend binary out of the VSIX and into GitHub Releases; SidecarManager downloads it once on first use per version — eliminating VS Marketplace AV false positive while preserving all behavior.

## Phases (36–37)
- Phase 36: sidecar-download — SidecarManager downloads from GitHub Releases on cache miss, verifies SHA256, shows progress UI, handles failure gracefully (completed 2026-04-01)
- Phase 37: release-pipeline — CI uploads platform binaries and SHA256 manifest as GitHub Release assets; bin/ excluded from VSIX (completed 2026-04-02)

## Key Accomplishments
- Cache-first binary resolution: globalStorage/<version>/ checked before any network call
- SHA256 checksum verification before extraction — integrity enforced
- vscode.window.withProgress download notification on first use per version
- Graceful failure: clear error + manual GitHub Releases download link
- VSIX shrinks from ~62 MB to ~1.5 MB; bin/ removed from package
- CI: gh release upload attaches tar.gz + checksums.sha256 as permanent Release assets
